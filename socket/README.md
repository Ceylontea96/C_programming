# 프로젝트 목적

1. 1:N 소켓 통신 구현
2. Multi-Thread 사용
3. Mutex를 통한 임계구역

##개발환경
- VIsual Studio 2019
- Microsoft Visual C++ 2019

## 프로그램 시퀀스
1. main을 통한 프로그램 시작
2. 프로그램 시작 시 Argument 입력값을 통한 server 및 client 구분
3. Server는 접속한 Client 관리 및 메시지 전달
4. Client는 특정 Server에 연결하여 Client간 채팅
